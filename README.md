# rag-nano

Minimum closed-loop librarian for RAG-driven assistants. Independent of any specific agent: an external orchestrator (chat agent, workflow agent, etc.) dispatches a query and receives ranked, fully-attributed knowledge results over a stable HTTP contract.

For setup, daily-use commands, configuration, and integration examples, see the active spec's quickstart:

→ [`specs/001-minimal-rag-loop/quickstart.md`](specs/001-minimal-rag-loop/quickstart.md)
